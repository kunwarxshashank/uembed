[Uembed](https://uembed.vercel.app/) video streaming app built with NEXTJS, Powered with VIDSTACK Player

#FEATURES
* Fast, Responsive, Secure & Easy To USE!
* Provides Daily Updates & Smooth Experience!
* Supports TMDB and IMDB id to fetch data based on it.
* Multi Language Choice to watch in your preferred language
* API Provides Subtitles in 50+ Languages!
